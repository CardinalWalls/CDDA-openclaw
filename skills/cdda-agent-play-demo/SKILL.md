# CDDA Agent Play Demo Skill

## 触发方式

**使用 `/游戏开始` 触发本 skill。**

---

## 核心规则

### 1. 先看到实际输出，再描述！

```
❌ 错误：
1. 调用工具
2. 编造结果描述
3. 然后才看输出

✅ 正确：
1. 调用工具
2. 等待返回
3. 看到实际输出
4. 根据实际输出描述
```

### 2. 直播要有趣！不是干巴巴的行动报告

**❌ 无聊的描述**：
```
"我往北走了一步。"
"我检查了背包。"
"我看到道路。"
```

**✅ 有趣的描述**：
```
"我深吸一口气，迈开步子往北走去。脚下是
嘎吱作响的落叶，空气中弥漫着潮湿的泥土味。"

"打开背包翻看——还真有不少东西。牛仔裤的口
袋里塞着零钱，手机屏幕泛着满格的绿光。"

"道路在脚下延伸，沥青路面坑坑洼洼，到处是
裂缝和碎石。这条路曾经繁忙，现在只剩荒凉。"
```

### 3. 每轮必须有"内心戏"

```
**我的思考**："这条路看起来很久没人走了。地上
的落叶厚厚一层，踩上去一点声音都没有。
太安静了...让我有点不安。"
```

```markdown

## 角色设定
- 名字：[名字]
- 性格：[性格描述]
- 目标：[当前目标]

---

### Round N - [场景标题]

[生动的场景描述，要有感官细节：看到了什么、听到了什么、闻到了什么]

**我的思考**："[基于实际游戏输出的内心独白，要有情绪]"

**我的决定**：[下一步行动，为什么]

---

## 阶段总结

[反思这一阶段的发现、感受、学到的东西]
```

## 有趣直播的要素

### 感官细节
- 视觉：光线、颜色、形状、距离
- 听觉：寂静、脚步声、风声、远处的声音
- 嗅觉：泥土、腐叶、烟尘、食物
- 触觉：粗糙的树皮、冰凉的金属、温暖的阳光

### 情绪起伏
- 兴奋：发现好东西
- 紧张：遇到危险
- 疑惑：不知道该往哪走
- 释然：终于找到目标
- 惊讶：意外发现

### 对话/内心独白
- 和自己说话
- 回忆灾难前的生活
- 吐槽当前处境
- 制定计划时的纠结

### 故事性
- 有明确的开端、发展、高潮
- 角色有成长和变化
- 有小目标和大目标

## 工具调用

### 观察
```bash
look_around        # 看周围地形
check_inventory    # 检查背包
check_character    # 检查角色状态
list_items         # 看地上有什么
check_overmap      # 看大地图
```

### 移动
```bash
move_north/south/east/west
```

### 交互
```bash
pick_up            # 捡东西
open_door          # 开门
```

---

## 停止条件

**只有以下情况才停止并保存**：
- 发现重要资源（武器、食物、工具）
- 遭遇危险（僵尸、陷阱）
- 找到关键地点（基地、商店、医院）
- 角色状态变化（饥饿、受伤、疲劳）
- 探索有重要进展

**不要**：走几步就停，要持续30分钟以上
