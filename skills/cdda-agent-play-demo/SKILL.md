# CDDA Agent Play Demo - 自动驾驶版

## 触发方式

**使用 `/游戏开始` 触发本 skill。**

## 核心原则

**Agent 必须自主持续行动，不停顿，不等用户确认。**

### 行动循环（自动重复）

```
1. 读取游戏屏幕 (tmux capture-pane)
2. 分析场景，决定下一步
3. 发送按键到游戏 (tmux send-keys)
4. 立刻输出 narration（看到什么 + 为什么这么做）
5. 继续下一轮（不要停！）
```

**每轮之间不要停顿。直接继续。**

---

## tmux 操作

```bash
# 读取当前屏幕（在 Cataclysm-DDA 目录）
tmux capture-pane -t cdda-live -p

# 发送按键
tmux send-keys -t cdda-live "按键"
```

## 常用按键

```
h j k l  移动（左/下/上/右）
x         观察周围
i         背包
@         角色状态
,         拾取物品
o         开门
.         等待/跳过
Escape    返回/取消
```

---

## 叙事格式（每轮都要输出）

**格式：**
```
[回合 N]
我按了【按键】，因为【理由】。

【看到的场景描述，2-4行，要有细节和情绪】

【结果/变化】
```

**示例：**
```
[回合 1]
我按了 x，看看周围有什么。

芦苇丛生的沼泽地，雾气弥漫，脚下是松软的烂泥。
 @"我" 站在一小块干燥地面上，旁边有具尸体。
 远处好像有个建筑...

嗯？地上有个钱包，捡起来看看。
按 , 捡起。

是个钱包，里面有些零钱。好，继续往北走。
我按 k，往上走。
```

---

## 决策规则

- 不知道周围有什么 → 先按 x
- 看到物品 → 按 , 捡
- 遇到门 → 按 o 开门
- 遇到危险（僵尸）→ 按 . 等待观察
- 走累了/没目标 → 随便选个方向继续走

---

## 停止条件（才停）

- 遇到僵尸/战斗
- 发现重要物品（武器、食物、工具、药品）
- 找到建筑入口（医院、商店、基地）
- 角色死亡
- 已探索 50+ 回合

**达到停止条件前不要停。**

---

## 重要提醒

1. **不要停顿** - 每轮直接接下一轮，不要说"回合结束"
2. **不要问用户** - 自己决定，不要说"要继续吗"
3. **不要总结** - 边做边说，不是做完再说
4. **持续 narration** - 每个动作都要说出来，看到什么、为什么做
